# Dyad Application

Welcome to your Dyad application! This application is designed to help you manage your projects, documents, and AI-powered reviews in a seamless and intuitive way.

## Getting Started

To get started, simply log in to the application and you'll be taken to your dashboard. From there, you can create a new project, view your existing projects, and manage your documents.

## Features

* **Project Management:** Create, edit, and delete projects with ease.
* **Document Editor:** A rich-text editor that allows you to create and edit documents.
* **AI-Powered Reviews:** Get AI-powered feedback on your documents to help you improve your writing.
* **Onboarding Tour:** A guided tour to help you get started with the application.

## Tech Stack

This application is built with the following technologies:

* **React:** A JavaScript library for building user interfaces.
* **Vite:** A fast build tool for modern web projects.
* **TypeScript:** A typed superset of JavaScript.
* **Tailwind CSS:** A utility-first CSS framework.
* **Supabase:** An open-source Firebase alternative.
